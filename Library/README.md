# Library 📚

A curated collection of reference materials, documentation, and knowledge sources.

---

## Categories

### Documentation
Technical docs, API references, user guides, and official documentation from tools you use.

### Books
Book summaries, notes, and takeaways from reading.

### Courses
Learning materials, course notes, and skill development resources.

### Research
Paper summaries, literature reviews, and domain knowledge.

---

## Adding New Notes

When adding to the library, consider including:

- Source URL or reference link
- Tags for categorization (e.g., `#api`, `#tutorial`, `#python`)
- Connection notes linking to related projects
