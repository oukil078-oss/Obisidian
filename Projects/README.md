# Projects 🚀

This folder contains all GitHub projects cloned to this system. Each project is automatically linked to the [[Profile]] note for graph visualization!

---

## How This Works 🔗

When I clone a new GitHub project, I'll:

1. Create a note using the `New Project.md` template
2. Link it to your [[Profile]] note
3. Add relevant tags and metadata
4. The Obsidian graph will automatically show connections!

---

## Graph Organization Tips 💡

To make your graph beautiful and organized:

- **Tags matter:** Use consistent tags like `#github`, `#python`, `#api`, `#webapp`
- **Link everything:** Connect related projects with links like [[Project A]] → [[Project B]]
- **Use callouts:** Add `:::callout{type="note"}` for important context
- **Mermaid diagrams:** Add flowcharts to show project architecture

---
