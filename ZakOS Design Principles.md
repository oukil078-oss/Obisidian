# ZakOS UI Design Principles

Avoid generic or "AI-made" aesthetics. Target premium, futuristic depth — glassmorphism + 3D-like layering.

## Reference Blend
- **Salesforce Invoices Dashboard**: dark surfaces, neon lime/green accents, split-panel list/detail, card metrics, dense but organized info hierarchy.
- **Glassmorphism SaaS (QuickBooks-style)**: frosted glass panels, blurred translucent surfaces, depth layering, soft drop shadows, ambient background glows.
- **Medical Dashboard (Dr+)**: clean squircle corners, good whitespace, card hierarchy, rounded UI elements.

## Mandatory Depth Cues
- Ambient background glows (subtle radial gradients that drift).
- Elevated glass layers with increased border definition and drop-shadow depth.
- Soft shadows with colored edges/glow on active elements.
- Smooth premium easing curves for all motion.
- Pulsing/breathing idle animations on interactive orbs and panels.

## Non-Negotiables
- Must not look generic, template-like, or obviously AI-generated.
- Must feel futuristic and modern, but not skeuomorphic.
- Must maintain high contrast and legibility.
- Must feel tactile — hover states should have real depth feedback.
